#### 软件架构

本开发使用SpringBoot+mybatis框架开发ModeAndView模式逻辑分明,大型web开发高并发

#### 开发环境如下:

1. Mysql 5.7
1. JDK 1.8
1. Redis

#### 开发工具如下:

1. IDEA 2018
1. Navicat for MySQL
1. Notepad++或其他编译文本工具

#### 安装教程

1.  配置JDK环境变量
2.  安装开发工具切运行Redis文件
3.  启动IDEA内置tomcat
